# 🎓 Student Task Manager

A simple and clean web application to manage daily student tasks efficiently.
Built using modern frontend technologies, this app helps students stay organized and productive.

---

## 🚀 Features

* ✅ Add new tasks
* 📅 Set due dates
* ✔️ Mark tasks as completed
* ❌ Delete tasks
* 📊 View total, completed, and pending tasks
* 💾 Data stored using Local Storage (persists after refresh)
* 📱 Responsive and user-friendly UI

---

## 🛠️ Tech Stack

* Frontend: React, HTML, CSS, JavaScript
* Storage: Browser Local Storage

---

---

## 🧠 Project Logic

1. User logs in using email and password
2. On successful login, user is redirected to Dashboard
3. User can:

   * Add tasks
   * Mark tasks as completed
   * Delete tasks
4. All tasks are stored in Local Storage
5. Tasks are loaded automatically on page refresh

---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/student-task-manager.git
```

2. Navigate to project folder:

```
cd student-task-manager
```

3. Install dependencies:

```
npm install
```

4. Run the project:

```
npm run dev
```
